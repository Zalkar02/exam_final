Программа должна запрашивать название текстового файла, который
нужно открыть. Необходимо посчитать количество строк, количество
слов и количество букв. Также необходимо вывести количество
повторений каждого слова, при этом надо учитывать, что в файле
могут быть дополнительные символы, такие как пробел ! " # $ % & '
( ) * + , - . / : ; < = > ? @ [ \ ] ^ _` { | } ~ Их не учитывать
при подсчете количества букв и слов.
Результат записать в новый файл с названием, состоящим из самого
повторяемого слова, например привет.txt
